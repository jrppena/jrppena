### 🎓 [TeachOS](https://www.teach-os.xyz) — AI lesson-plan generation for Filipino teachers · `LIVE`
[![Frontend](https://img.shields.io/badge/Frontend-Repo-181717?style=flat&logo=github&logoColor=white)](https://github.com/jrppena/teach-os-fe)
[![Backend](https://img.shields.io/badge/Backend-Repo-181717?style=flat&logo=github&logoColor=white)](https://github.com/jrppena/teach-os-be)

`React 19` · `TypeScript` · `FastAPI` · `SQLAlchemy 2.0` · `asyncpg` · `NeonDB` · `Firebase Auth` · `Gemini 2.5` · `Grok 3` · `Docker` · `AWS` · `Vercel`

A provider-agnostic AI platform that generates DepEd MATATAG/ILAW-compliant lesson plans. Built over raw `httpx` — no vendor SDKs — dispatching to both Gemini 2.5 Flash and Grok 3, with teachers bringing their own API key.
- Structured-prompt + server-side post-validation pipeline overrides **5+ deterministic fields**, making format non-compliance impossible regardless of model drift
- Fully async REST API across **9 FastAPI endpoints** on SQLAlchemy 2.0 + asyncpg — zero blocking I/O
- Type-safe React 19 + TanStack Query v5 + Zod frontend across **113 source files**
- Dockerized backend on AWS, frontend on Vercel; stateless auth via Firebase Admin SDK bearer tokens
